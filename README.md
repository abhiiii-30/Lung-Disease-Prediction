# 🫁 Lung Disease Prediction using AI

## 📌 Overview
This project focuses on predicting lung diseases using Artificial Intelligence (AI), Machine Learning (ML), and Deep Learning (DL). It analyzes patient data and medical images like chest X-rays to detect diseases early.

---

## 🎯 Objectives
- Early detection of lung diseases
- Improve diagnostic accuracy
- Assist healthcare professionals
- Reduce manual errors

---

## 🧠 Diseases Detected
- Pneumonia
- Tuberculosis
- COPD
- Lung Cancer

---

## ⚙️ Technologies Used
- Python
- Scikit-learn
- TensorFlow / PyTorch
- OpenCV
- Pandas, NumPy
- Matplotlib

---

## 🏗️ System Architecture
1. Data Collection (X-ray, CT Scan, Patient Data)
2. Data Preprocessing
3. Feature Extraction
4. Model Training
5. Prediction
6. Result Visualization

---

## 📊 Model Used
- Random Forest Classifier
- CNN (for image-based prediction)

---

## 📈 Results
- Accuracy: ~90-93%
- Precision: ~92%
- Recall: ~91%
- Fast prediction (~2-3 seconds)

---

## 🚀 Features
- Predicts lung disease from input data
- Supports clinical decision making
- Scalable and efficient system
